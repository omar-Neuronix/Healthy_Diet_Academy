# جدول شرح حقول لوحة التحكم (public/admin.html)

> الملف: `public/admin.html`

| القسم | الحقل | النوع/الوصف | القيم المتوقعة | الهدف/ماذا يفعل |
|---|---|---|---|---|
| الصلاحية | **adminToken** | Password (Bearer token) | مثال: `Bearer ADMIN_KEY` أو فقط `ADMIN_KEY` | يستخدم للتحقق أن المستخدم “مسؤول” لإرسال الطلبات الإدارية (مثل إضافة الشهادة وتحديث لينك الامتحان). |
| الصلاحية | **API Base** | Readonly | يتم ضبطه تلقائيًا | عرض عنوان API المستخدم من الصفحة. |
| الشهادات | **certId** | Text | مثل: `HDA-2026-05` | كود الشهادة الفريد المستخدم للتأكد/البحث عنها. |
| الشهادات | **certificateImage** | Text | اسم ملف داخل المشروع (مثل `3.png`) أو رابط كامل (URL) | صورة الشهادة التي ستظهر في صفحة التحقق. |
| الشهادات | **studentName** | Text | اسم الطالب/الخريج | يظهر داخل بيانات الشهادة. |
| الشهادات | **courseName** | Text | اسم الكورس/الدبلومة | يظهر داخل بيانات الشهادة. |
| الشهادات | **issueDate** | Date | تاريخ (YYYY-MM-DD) | تاريخ إصدار الشهادة. |
| الشهادات | **score** | Number | من 0 إلى 100 | درجة الطالب (للتحقق والعرض). |
| الشهادات | **grade** | Text | مثل: `امتياز` / `جيد جدًا` | التقدير العام للشهادة. |
| الشهادات | **issuer** | Text | اسم الجهة/المرجع | الجهة المانحة أو المرجع الذي صدر عنه الاعتماد. |
| الشهادات | **stamp** | Text | مثل: `معتمدة رسميًا` | نص الختم/الاعتماد الظاهر. |
| الشهادات | **verificationCode** | Text | مثل: `VER-2026-005` | كود التحقق المطبوع على الشهادة (للتحقق والتوثيق). |
| الشهادات | **isValid** | Select | `true` أو `false` | صلاحية الشهادة:
- `true` = صالحة (تظهر في verify)
- `false` = غير صالحة (ترفض التحقق). |
| الشهادات | **إرسال/إضافة شهادة (addCertBtn)** | Button | — | يرسل بيانات الشهادة عبر:
`POST /api/cert/add` (محمي بـ ADMIN_KEY). |
| الشهادات | **مسح الحقول (clearBtn)** | Button | — | يمسح قيم الحقول من الفورم. |
| الامتحانات | **activeExamLink** | Text | URL مثل: `https://forms.microsoft.com/...` | رابط الامتحانات الذي سيظهر في index.html. |
| الامتحانات | **تحديث رابط الامتحانات (updateExamLinkBtn)** | Button | — | يرسل تحديث الرابط عبر:
`POST /api/admin/exam-link` (محمي بـ ADMIN_KEY). |
| المخرجات | **status** | Pre (رسالة) | — | يعرض رسائل نجاح/فشل أثناء تنفيذ الطلبات. |
